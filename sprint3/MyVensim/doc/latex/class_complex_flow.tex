\doxysubsection{Detailed Description}
Complex flow class derived from flow. 

Despite its name, it represents the simplest interaction beetween systems, subtracting a constant stock from the source \doxylink{class_system}{System} and adding it to the target one. Its only purpose is to implement the pure virutal method \doxylink{class_complex_flow_a86d4a74a9c3f8c962d85e559311e0dcb}{run()}, from flow.
