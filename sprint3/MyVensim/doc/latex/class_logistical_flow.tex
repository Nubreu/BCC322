\doxysubsection{Detailed Description}
Logistical flow class derived from flow. 

Represents a logistical interaction beetween systems, flow is sort of inverted, taking the stock from the target \doxylink{class_system}{System} and transfering it to the source \doxylink{class_system}{System}. Its only purpose is to implement the pure virutal method \doxylink{class_logistical_flow_aa380b703376d79abc5de059b18540e41}{run()}, from flow.
